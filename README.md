# DealFlow CRM

CRM en construcción para gestión de deals comerciales. Permite registrar, seguir y actualizar el estado de oportunidades de negocio a lo largo del pipeline de ventas.

## Stack

- Frontend: HTML + JavaScript vanilla
- Backend: Node.js REST API
- CI/CD: GitHub Actions

## Módulos

- [x] Gestión de deals (pipeline)
- [x] Envío de contratos (DocuSign)
- [ ] Contactos
- [ ] Reportes
